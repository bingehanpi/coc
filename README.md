# coc
联赛排名

运行前请检查库是否齐全~

1.运行前请登录https://developer.clashofclans.com/ ,自行创建账号和设立官方api,因为是外网,有科学上网影响这里提供ip查询:

A.自行查找本地ip并完成官方api的创建

B.进入test.py文件,在API_TOKEN后面填入你的apikey,并运行

C.根据结果找到你的在网ip地址,根据这个ip地址重新创建api


注:后续步骤运行前记得删除json和xlsx文件噢

2.进入clans_get.py文件运行,理论上api正确的情况下你将获得一个json文件,后面代码运行的基础就是这个(运行错误检查api是否正确,ip地址是否变化)

3.进入json_analysis.py文件运行,理论上运行结果是正确的,如果不正确说明部落联赛处于一种真空期,稍等1小时运行(当然也可能是真错了)

4.进入json_excel_raw.py文件运行,在步骤3运行正确的情况下,你将获得一个cwl_raw_readable.xlsx文件,这是当前联赛所有部落的进攻和防守数据

5.进入json_excel_only.py文件运行,你将获得一个cwl_our_clan_only.xlsx文件,这是查询部落有关的进攻和防守数据,后续步骤是进行数据分析

6.进入clans_last.py文件运行,你将获得一个总体数据分析表格
